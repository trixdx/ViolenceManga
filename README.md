# Violence — Manga Reader

Приложение для чтения манги с фиолетовой темой. Данные загружаются через [MangaDex API](https://api.mangadex.org).

## Возможности

- Поиск, жанры, популярное и новые главы (MangaDex API)
- Читалка: вертикальный/горизонтальный режим, навигация по страницам и главам
- Авто-закладки и «Продолжить чтение»
- Профиль с уровнями, XP и временем чтения
- 10 достижений с уведомлениями
- Избранное, списки (Читаю / В планах / Прочитано / Брошено)
- Настройки: тема, язык, NSFW-фильтр, сортировка глав
- Горячие клавиши (`Ctrl+K`, `?`, стрелки)
- URL-маршруты (`#/manga/id`, `#/search`, `#/genre`)

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```
